# My Smart Mirror

## Functionalities:
* Displays the current date and time
* Displays Weather of a specified city
  * Use the edit button to change the city
* Can tell a dad joke to help you smile :)

## Deyployment:
* Deployed with Netlify: https://tvor-smart-mirror.netlify.app/
